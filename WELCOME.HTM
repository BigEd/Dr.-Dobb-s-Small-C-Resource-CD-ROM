Dr. Dobb's Small-C Resource CD-ROM

Welcome to Dr. Dobb's Small-C Resource CD-ROM, the definitive collection of Small-C related information and source code. This CD-ROM includes the full text to James Hendrix's book A Small-C Compiler: Language, Usage, Theory, and Design, selected articles on Small-C from Dr. Dobb's Journal magazine, and Small-C implementations for a number of processor platforms, including the 8080, Z80, 6502, and others.

The text to both the Small-C Compiler book and Dr. Dobb's Journal articles is provided in HTML format. To go to either of these text areas, select the appropriate link below.

Upon selecting one of the above links, you will go to the Table of Contents for one of the two text areas. Content entries are linked to individual chapters or articles.

  • At the end of each chapter of the book, you can go to the next chapter or return to the book's Table of Contents.
  • At the end of each article, you can return to the article Table of Contents.
  • At the end of each Table of Contents, you can return to this page.

The Small-C compilers and related source code are available in the SMALLC directory, located at the root directory level. Within the SMALLC directory is a brief overview of the directory contents (see README.ASC). Detailed documentation and readme files for the various implementations are included in the appropriate subdirectories.

To view the book and articles, you will need a HTML capable browser.
