# Dr.-Dobb-s-Small-C-Resource-CD-ROM
[Mirror] Definitive collection of Small-C related information and source code. Includes the full text to James Hendrix's book "A Small-C Compiler: Language, Usage, Theory, and Design," selected articles on Small-C from "Dr. Dobb's Journal" magazine, and Small-C implementations for a number of processor platforms, including the 8080, Z80, 6502.
